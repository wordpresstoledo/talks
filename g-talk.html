✍️ Why Gutenberg?

Gutenberg is a take on a new WYSIWYG post editor for WordPress. It is named after Johannes Gutenberg, who invented a printing press with movable type more than 500 years ago.

⭐ Goals

Their goal is to make working in WordPress easier, especially for those just starting.

See the sandbox example.

👑 Meta Boxes Rule

For a long time, meta boxes were behind all custom fields in our websites. You'd find them in ...

  • 🔌 Plugins
  • 👍 Themes

    • 🌩️ TinyMCE

    Default text editor is the defacto default interface everyone uses.

    This editor has found its way into many CMS, with not the best user interface. For many, this is the CMS.

    To implement customization, we utilize shortcodes, visual editors, and HTML to make things work.

    [/vc_column_text][vc_row_inner][vc_column_inner width="1/4"][/vc_column_inner][vc_column_inner width="1/2" css=".vc_custom_1507469888235{background-color: rgba(255,255,255,0.6) !important;}"]


    It's not very pretty or readable.

    WordPress is changing the model.
